# Parity Test Scaffold

This directory stores automated and manual parity artifacts used to verify Qt rewrite behavior against VB iBMSC.

## Layout
- `io_golden/`: input and expected output chart corpus for parser/save diff checks.
- `interaction/`: deterministic interaction scripts and expected state logs.
- `snapshots/`: screenshot baselines for UI parity checks.

## Current Status
Scaffold only. Test runners and fixtures will be added in subsequent parity phases.
