# Ableton Theme Spec

## Overview
Replace the current blue-tinted dark theme with an Ableton Live-inspired flat dark palette using warm orange accents.

## Requirements

### Colors
- Background hierarchy: `#0a0a0a` (base) → `#1a1a1a` (surface) → `#2a2a2a` (panel) → `#333333` (card) → `#3d3d3d` (hover)
- Primary accent: `#ff764d` (Ableton orange)
- Accent hover: `#ff8f6d`
- Accent muted: `#994a30`
- Text primary: `#ffffff`
- Text secondary: `#999999`
- Text disabled: `#555555`
- Border primary: `#333333`
- Border faint: `#222222`
- Success: `#7ec856`, Warning: `#d4a843`, Error: `#d94452`

### Geometry
- Border radius: 0–2px (flat, minimal rounding)
- Spacing: keep existing scale (4/8/12/16/24)

### Typography
- Keep existing font families
- Section headers: uppercase, letter-spacing 1.5px, font-size 10px

### Animation
- Keep existing animation durations
